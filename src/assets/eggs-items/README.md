# Eggs & Items PNG Images

Drop egg and item PNG images here.

## Required Images:

1. **royal-jelly.png** - The egg/royal jelly image (REQUIRED!)
   - This is what the player drags and drops into hive slots
   - Will be displayed 10 times (10 eggs available)
   - Displayed at 60x60px

## Image Requirements:

- **Format**: PNG with transparent background
- **Size**: Recommended 128x128px to 256x256px
- **Style**: Should look like a glowing egg or royal jelly container

## Optional Images (for future):

- special-jelly.png - For rare egg variants
- honey-jar.png - For other game mechanics
- pollen.png - For future features

## Fallback:

If royal-jelly.png is missing, will show 🥚 emoji instead.
